## bluebell
* bluebell是一个基于gin框架和Vue框架搭建的前后端分离的web项目。
## 技术栈 
* gin 框架 
* zap 日志库
* viper 配置文件管理 
* JWT 认证
* air go程序热加载
* swagger 生成接口文档 
* ratelimit 令牌桶限流
* sqlx 操作 MySQL 数据库
* go-redis 操作 Redis 数据库
* go web 应用部署
